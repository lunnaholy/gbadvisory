### Небольшое предысловие

Первая версия была написана на React, но размер одного чанка перевалил за 500kb и я решил оставить эту идею :)

Я сделал адаптив, используя правила @media, но решил не запариваться с nav-листом и просто отключил его на таблетах и мобильных устройствах. Вообще, так не нужно делать (я так думаю), но прописывать JS-анимки вручную для мобильного навбара я тоже не хотел.

Для максимальной оптимизации загрузки я принял решение ужать все файлы с помощью minifier'a. Исходные версии файлов хранятся в тех же директориях, только в их названии нет суффикса "min".

Соответственно, entry-point - тоже index.min.html
